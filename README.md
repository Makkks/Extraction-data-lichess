# Extraction-data-lichess

## Présentation du projet
Ce projet présente une méthode d’extraction de données
en streaming sequentiel (torrent).
L’objectif est de proposer une approche reproductible
basée sur des outils open source.

## Technologies utilisées
- Python
- Extraction séquentiel
- qBittorrent
- Stockage CSV

## Structure du projet
- notebooks/ : notebook d’extraction des données
- data/ : sample des données traités
- docs/ : documentation et tutoriels
