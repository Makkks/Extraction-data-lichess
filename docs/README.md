# Tutoriel qBittorrent

## Description

Ce dossier contient un **document tutoriel** expliquant l’utilisation de qBittorrent
dans le cadre de ce projet.

Le tutoriel a pour objectif de décrire, de manière pédagogique,
comment qBittorrent est utilisé comme outil d’extraction de données
à partir de sources open source.

---

## Remarque

Ce document est un **tutoriel explicatif**.
Il ne s’agit pas d’un outil automatisé, mais d’un support de compréhension
permettant de reproduire la démarche d’extraction.
